# Initiation Objet avec Typescript

## Récupération du projet

`git clone https://github.com/dacodemaniak/abap_ts_object.git`

## Installation des dépendances

```shell
cd abap_ts_object
npm i
```

## Exécuter

```shell
npm run start
````

## Tester
```shell
npm run test
npm run test:watch
npm run test:coverage
```
